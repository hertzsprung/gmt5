#include "common_options.txt"
.TH GMTLOGO 1 MANDATE GMT_STRING "Generic Mapping Tools"
.SH NAME
gmtlogo \- Adding a GMT graphics logo overlay to an illustration
.SH SYNOPSIS
BD(gmtlogo) IT(dx) IT(dy) [ OPT(G)IT(fill) ] [ OPT(W)[IT(pen)] ] >> IT(plot.ps)
.SH DESCRIPTION
This scrips appends the BD(GMT) logo as an overlay to an "open" PostScript file.
The logo is 2 inches wide and 1 inch high and will be positioned relative to the current plot origin.
#include "explain_commonitems.txt"
.SH REQUIRED ARGUMENTS
.TP
IT(dx),IT(dy)
Sets the lower-left corner of the logo relative to current plot origin.
.SH OPTIONAL ARGUMENTS
.TP 
OPT(G)IT(fill)
Select color or pattern for filling the underlying box [Default is no fill].
.TP
OPT(W)[IT(pen)]
Set pen attributes for the outline of the box [Default is no outline].
.SH "SEE ALSO"
.IR gmt (1),
.IR gmtcolors (5),
.IR psimage (1)
