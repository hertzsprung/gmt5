.TH GRDSPOTTER 1 MANDATE GMT_STRING "Generic Mapping Tools"
.SH NAME
grdspotter \- Create CVA image directly from gravity or bathymetry grids 
.SH SYNOPSIS
BD(grdspotter) [IT(grdfile)] OPT(E)IT(rotfile) 
OPT(G)IT(CVAgrid) GMT_I_OPT 
GMT_Rgeo_OPT [ OPT(A)IT(agegrid) ] [ OPT(B)IT(n_try) ] [ OPT(D)IT(DIgrid) ] 
[ OPT(L)IT(IDgrid) ] [ OPT(M) ] [ OPT(N)IT(upper_age) ] [ OPT(P)IT(PAgrid) ] [ OPT(Q)IT(IDinfo) ] [ OPT(S) ] 
[ OPT(T)BD(t)|BD(u)IT(fixed_val) ] [ GMT_V_OPT ] [ OPT(Z)IT(z_min)[/IT(z_max)[/IT(z_inc)]] ] [ GMT_r_OPT ]
.SH DESCRIPTION
BD(grdspotter) reads a grid file with residual bathymetry or gravity
and calculates flowlines from each node that exceeds a minimum value
using the specified rotations file.
These flowlines are then convolved with the volume of the prism represented by
each grid node and added up to give a Cumulative Volcano Amplitude grid (CVA).
.br
	No space between the option flag and the associated arguments.  Use upper case for the
option flags and lower case for modifiers.
.SH REQUIRED ARGUMENTS
.TP
IT(grdfile)
Data grid to be processed, typically residual bathymetry or free-air anomalies.
#include "euler.txt"
.TP
OPT(G)
Specify name for output CVA grid file.
#include "explain_-I.txt"
#include "explain_-Rgeo.txt"
.SH OPTIONAL ARGUMENTS
.TP
OPT(A)IT(agegrid)
Supply a crustal age grid that is co-registered with the input data grid.  These ages
become the upper ages to use when constructing flowlines [Default extend flowlines back
to oldest age found in the rotation file; but see OPT(N)].
.TP
OPT(B)IT(n_try)
Get IT(n_try) bootstrap estimates of the maximum CVA location; the longitude and latitude
results are written to stdout [Default is no bootstrapping].  Cannot be used with OPT(M).
.TP
OPT(D)IT(DIgrid)
Use flowlines to determine the maximum CVA encountered along each flowline and create a Data
Importance (DI) grid with these values at the originating nodes.
.TP
OPT(L)IT(IDgrid)
Supply a co-registered grid with seamount chain IDs for each node.  This option requires that
you also use OPT(Q).
.TP
OPT(M)
Do not attempt to keep all flowlines in memory when using OPT(D) and/or OPT(P).  Should you run out of memory you can
use this option to compute flowlines on-the-fly.  It will be slower as we no longer
can reuse the flowlines calculated for the CVA step.  Cannot be used with OPT(B) or the
multi-slice mode in OPT(Z).
.TP
OPT(N)IT(upper_age)
Set the upper age to assign to nodes whose crustal age is unknown (i.e., NaN) [no upper age].
Also see OPT(A).
.TP
OPT(P)IT(PAgrid)
Use flowlines to determine the flowline age at the CVA maximum for each node and create a Predicted
Age (PA) grid with these values at the originating nodes.
.TP
OPT(Q)IT(IDinfo)
Either give (1) a single ID to use or (2) the name of a file with a list of IDs to use [Default uses all IDs].
Each line would be TAG ID [w e s n].  The IT(w/e/s/n) zoom box is optional; if specified it means
we only trace the flowline if inside this region [Default uses region set by OPT(R)].  Requires OPT(L).
.TP
OPT(S)
Normalize the resulting CVA grid to percentages of the CVA maximum.  This also normalizes the DI grid (if requested).
.TP
OPT(T)BD(t)|BD(u)IT(fixed_val)
Selects ways to adjust ages; repeatable.  Choose from OPT(Tt) to
truncate crustal ages given via the OPT(A) option that exceed the upper age set with OPT(N) [no truncation],
or OPT(Tu)IT(fixed_val) which means that
after a node passes the test implied by OPT(Z), we use this IT(fixed_val) instead in the calculations.
[Default uses individual node values].
#include "explain_-V.txt"
.TP
OPT(Z)IT(z_min)[/IT(z_max)[/IT(z_inc)]]
Ignore nodes with z-values lower than IT(z_min) [0] and optionally larger than IT(z_max) [Inf].
Give IT(z_min/z_max/z_inc) to make separate CVA grids for each IT(z)-slice [Default makes one CVA grid].
Multi-slicing cannot be used with OPT(M).
#include "explain_nodereg.txt"
#include "explain_help.txt"
.SH EXAMPLES
To create a CVA image from the Pacific topography grid Pac_res_topo.nc,
using the DC85.d Euler poles, and only output a grid for the specified domain, run
.br
.sp
BD(grdspotter) Pac_res_topo.nc OPT(E)DC85.d OPT(G)CVA.nc OPT(R)190/220/15/25 OPT(I)2BD(m) OPT(N)145 OPT(Tt) OPT(V)
.br
.sp
This file can then be plotted with BD(grdimage).
