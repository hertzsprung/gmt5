.SH NAME
BD(pspolar)\ - Plot polarities on the inferior focal half\-sphere on maps
.SH SYNOPSIS
.br
pspolar [ IT(table) ] GMT_J_OPT GMT_Rgeo_OPT
OPT(D)IT(lon)/IT(lat) OPT(M)IT(size) OPT(S)IT(<symbol><size>)
[ GMT_B_OPT ] [ OPT(C)IT(lon)/IT(lat)[/IT(dash_width)/IT(pointsize)] ] 
[ OPT(F)IT(color) ] [ OPT(G)IT(fill) ] [ OPT(g)IT(fill) ] 
[ GMT_K_OPT ] [ OPT(L) ] [ OPT(N) ] [ GMT_O_OPT ] [ GMT_P_OPT ] 
[ OPT(s)IT(Half-size)[BD(v)[[IT(v_width)/IT(h_length)/IT(h_width)/IT(shape)]][BD(g)[IT(color)]][BD(l)]
[ OPT(T)IT(angle)/IT(form)/IT(justify)/IT(fontsize) ] [ OPT(t)IT(pen) ]
[ GMT_U_OPT ] [ GMT_V_OPT ] [ OPT(W)IT(pen) ]
[ GMT_X_OPT ] [ GMT_Y_OPT ] [ GMT_c_OPT ] [ GMT_h_OPT ] [ GMT_i_OPT ]
.SH DESCRIPTION
BD(pspolar) reads data values from IT(files) [or standard input]
and generates IT(PostScript) code that will plot stations on focal mechanisms
on a map.
The IT(PostScript) code is written to standard output.
.br
.sp
Parameters are expected to be in the following columns:
.RS
.TP
BD(1),BD(2),BD(3):
station_code, azimuth, take-off angle
.TP
BD(4):
polarity:
.br
- compression can be c,C,u,U,+
.br
- rarefaction can be d,D,r,R,-
.br
- not defined is anything else
.RE
#include "explain_commonitems.txt"
.SH REQUIRED ARGUMENTS
#include "explain_intables.txt"
#include "explain_-J.txt"
#include "explain_-Rgeo.txt"
.TP
OPT(D)longitude/latitude)
Maps the bubble at given longitude and latitude point.
.TP
OPT(M)size)
Sets the size of the beach ball to plot polarities in.  IT(Size) is in inch
(unless BD(c), BD(i), BD(m), or BD(p) is appended).
.TP
OPT(S)<symbol_type><size>)
Selects IT(symbol_type) and symbol IT(size).  
Size is in inch (unless BD(c), BD(i), BD(m), or BD(p) is appended).
Choose symbol type from
st(IT(a))r, (IT(c))ircle, (IT(d))iamond, (IT(h))exagon,
(IT(i))nverted triangle, (IT(p))oint, (IT(s))quare, (IT(t))riangle,
(IT(x))cross.
.SH OPTIONAL ARGUMENTS
#include "explain_-B.txt"
.TP
OPT(C)
Offsets focal mechanisms to the latitude and longitude specified in the last
two columns of the input file.
.TP
OPT(E)fill)
Selects filling of symbols for stations in extensive quadrants.
Set the color [Default is 250].
If OPT(E)IT(fill) is the same as OPT(F)IT(fill), use OPT(e) to outline.
.TP
OPT(e)[pen])
Outline symbols in extensive quadrants using IT(pen) or the default pen (see
OPT(W)).
.TP
OPT(F)IT(fill)
Sets background color of the beach ball. Default is no fill.
.TP
OPT(G)IT(fill)
Selects filling of symbols for stations in compressional quadrants.
Set the color [Default is black].
#include "explain_-K.txt"
.TP
OPT(N)
Does BD(not) skip symbols that fall outside map border [Default plots points inside border only].
#include "explain_-O.txt"
#include "explain_-P.txt"
.TP
OPT(T)IT(angle/form/justify/fontsize in points)
To write station code. [Default is 0.0/0/5/12].
#include "explain_-U.txt"
#include "explain_-V.txt"
.TP
OPT(W)IT(pen)
Set current pen attributes 
 [Defaults: width = default, color = black, style = solid].
#include "explain_-XY.txt"
#include "explain_-c.txt"
.TP
OPT(f)[pen])
Outline the beach ball using IT(pen) or the default pen (see OPT(W)).
.TP
OPT(g)[pen])
Outline symbols in compressional quadrants using IT(pen) or the default pen
(see OPT(W)).
.TP
OPT(h)
Use special format derived from HYPO71 output
#include "explain_-icols.txt"
.TP
OPT(s)IT(half\-size)/[BD(V)[IT(v_width/h_length/h_width/shape)]][BD(G)IT(color)][BD(L)]
Plots S polarity azimuth.
S polarity is in last column.
It may be a vector (BD(V) option) or a segment. 
Give half\-size,v_width,h_length,h_width in inch
(unless BD(c), BD(i), BD(m), or BD(p) is appended).
[BD(L)] option is for outline.
.TP
OPT(t)IT(pen)
Set pen color to write station code. Default uses the default pen (see
OPT(W)).
