#include "common_options.txt"
.TH GRDPASTE 1 MANDATE GMT_STRING "Generic Mapping Tools"
.SH NAME
grdpaste \- Paste together two IT(.nc) files along a common edge.
.SH SYNOPSIS
BD(grdpaste) IT(file_a.nc file_b.nc) OPT(G)IT(outfile.nc) [ GMT_V_OPT ] [ GMT_f_OPT ]
.SH DESCRIPTION
BD(grdpaste) will combine IT(file_a.nc) and IT(file_b.nc) into IT(outfile.nc) by pasting
them together along their common edge.  Files IT(file_a.nc) and IT(file_b.nc) must have the same dx, dy and have one edge
in common.  If in doubt, check with BD(grdinfo) and use BD(grdcut) and/or BD(grdsample)
if necessary to prepare the edge joint.  For geographical grids, use OPT(f) to handle periodic longitudes.
.SH REQUIRED ARGUMENTS
.TP
IT(file_a.nc)
One of two files to be pasted together.
.TP
IT(file_b.nc)
The other of two files to be pasted together.
.TP
OPT(G)IT(outfile.nc)
The name for the combined output.
.SH OPTIONAL ARGUMENTS
#include "explain_-V.txt"
#include "explain_-f.txt"
#include "explain_help.txt"
#include "explain_grd_inout.txt"
.SH EXAMPLES
Suppose file_a.nc is 150E - 180E and 0 - 30N, and file_b.nc is 150E - 180E, -30S - 0, then you can
make outfile.nc which will be 150 - 180 and -30S - 30N by: 
.sp
BD(grdpaste) file_a.nc file_b.nc OPT(G)outfile.nc OPT(V) OPT(f)
