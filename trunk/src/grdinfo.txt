#include "common_options.txt"
.TH GRDINFO 1 MANDATE GMT_STRING "Generic Mapping Tools"
.SH NAME
grdinfo \- Get information about the contents of a 2-D grid file
.SH SYNOPSIS
BD(grdinfo) IT(grdfiles) [ OPT(C) ] [ OPT(F) ] 
[ OPT(I)[IT(dx)[/IT(dy)]] ] [ OPT(L)[BD(0)|BD(1)|)2)] ] 
[ OPT(M) ] [ GMT_Rgeo_OPT ] [ OPT(T)IT(dz) ] [ GMT_V_OPT ] [ GMT_f_OPT ]
.SH DESCRIPTION
BD(grdinfo) reads a 2-D binary grid file and reports various statistics for
the (IT(x),IT(y),IT(z)) data in the grid file(s).  The output information contains the
minimum/maximum values for IT(x), IT(y), and IT(z), where the min/max of IT(z) occur, the
IT(x)- and IT(y)-increments, and the number of IT(x) and IT(y) nodes, and [optionally] the
mean, standard deviation, and/or the median, L1 scale of IT(z), and number of
nodes set to NaN.
.SH REQUIRED ARGUMENTS
.TP
IT(grdfile)
The name of one or several 2-D grid files.
(See GRID FILE FORMATS below.)
.SH OPTIONAL ARGUMENTS
No space between the option flag and the associated arguments.
.TP
OPT(C)
Formats the report using tab-separated fields on a single line.  The output is
IT(w e s n z0 z1 dx dy nx ny )[ IT(x0 y0 x1 y1) ] [ IT(med scale) ] [IT( mean std rms)] [IT(n_nan)].
The data in brackets are output only if the corresponding options
OPT(M), OPT(L1), OPT(L2), and OPT(M) are used, respectively.
If the OPT(I) option is used, the output format is instead
IT(NF w e s n z0 z1), where IT(NF) is the total number of grids read and IT(w e s n) are rounded off (see OPT(I)).
.TP
OPT(F)
Report grid domain and x/y-increments in world mapping format [Default is generic].
Does not apply to the OPT(C) option.
.TP
OPT(I)[IT(dx)[/IT(dy)]]
Report the min/max of the region to the nearest multiple of IT(dx) and IT(dy),
and output this in the form OPT(R)IT(w/e/s/n) (unless OPT(C) is set).
To report the actual grid region, select OPT(I-).  If no argument is given
then we report the grid increment in the form OPT(I)IT(xinc/yinc).
.TP
OPT(L)[BD(0)|BD(1)|)2)]
.TP
OPT(L0)
Report range of z after actually scanning the data, not just reporting what the header says.
.TP
OPT(L1)
Report median and L1 scale of z (L1 scale = 1.4826 * Median Absolute Deviation (MAD)).
.TP
OPT(L2)
Report mean, standard deviation, and rms of z.
.TP
OPT(M)
Find and report the location of min/max z-values, and count and report the number of nodes
set to NaN, if any.
#include "explain_-R.txt" 
Using the OPT(R) option will select a subsection of the input grid(s). If this subsection exceeds the
boundaries of the grid, only the common region will be extracted.
.TP
OPT(T)IT(dz)
Determine min and max z-value, round off to multiples of IT(dz), and report as the text string
OPT(T)IT(zmin/zmax/dz) for use by BD(makecpt).
