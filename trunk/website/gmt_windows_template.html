GMT for Windows


We offer one required and one optional Windows Installers for those who do not wish to compile from source:

Required: Choose either the 32-bit or 64-bit GMT installer to install GMT version GMT_VERSION with all executables (including supplements and GDAL/OGR utilities), the netCDF, PCRE and GDAL DLL libraries, the gawk utility, the complete GSHHS version GSHHS_VERSION coastlines, rivers, and borders, all HTML documentation, and the example batch scripts with test data [73 Mb].

Optional: The PDF installer will install the complete documentation in PDF format [19 Mb].

Should you have additional needs for source code or tutorial data sets, etc., you may download the complete GMT distribution as a bzipped tarball from any of our GMT ftp sites and extract the files with a suitable extractor (such as 7-zip).
