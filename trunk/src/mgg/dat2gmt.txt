.TH DAT2GMT l DATE
.SH NAME
dat2gmt \- Convert an ASCII file to a binary gmt file
.SH SYNOPSIS
\fBdat2gmt\fP \fIgmtfile\fP
.SH DESCRIPTION
dat2gmt reads an ASCII version of a gmt file (made by gmt2dat) from standard
input and creates a binary gmt file.
.SH EXAMPLES
To create a gmt file from the ASCII file c2104_fixed.ascii, try
.br
.sp
dat2gmt c2104.gmt < c2104_fixed.ascii
.SH "SEE ALSO"
.IR gmt2dat (l),
.IR gmt (l)
