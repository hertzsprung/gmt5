#include "common_options.txt"
.TH GRDPMODELER 1 MANDATE GMT_STRING "Generic Mapping Tools [spotter supplement]"
.SH NAME
grdpmodeler \- Evaluate a plate model on a geographic grid
.SH SYNOPSIS
BD(grdpmodeler) IT(agegrdfile) OPT(E)IT(rot_file)
OPT(G)IT(outgrdfile) OPT(S)BD(a)|BD(d)|BD(r)|BD(w)|BD(x)|BD(y)|BD(X)|BD(Y) [ OPT(F)IT(polygonfile) ] 
[ OPT(T)IT(age) ] [ GMT_V_OPT ] [ GMT_b_OPT ] [ GMT_h_OPT ] [ GMT_i_OPT ] [ GMT_colon_OPT ]
.SH DESCRIPTION
BD(grdpmodeler) reads a geographical age grid and a plate motion model
and evaluates one of several model predictions.  Optionally, the user may supply a clipping polygon
in multiple-segment format; then, only the part of the grid inside the polygon is used
to determine the model prediction; the remainder of the grid is set to NaN.
#include "explain_commonitems.txt"
.SH REQUIRED ARGUMENTS
.TP
IT(ingrdfile)
Name of a grid file in geographical (lon, lat) coordinates with ages in Myr.
#include "euler.txt"
.TP
OPT(G)IT(outgrdfile)
Name of output grid.  This is the grid with the model predictions given the specified
rotations.
.TP
OPT(S)BD(a)|BD(d)|BD(r)|BD(w)|BD(x)|BD(y)|BD(X)|BD(Y)
Type of model prediction.  Choose from BD(a) for plate motion azimuth, BD(d) for distance between
current locations and their origin at the ridge (in km), BD(r) for plate motion rate (in mm/yr),
BD(w) for plate rotation rate (degree/Myr), BD(x) change in longitude relative to location of crust formation,
BD(y) change in latitude relative to location of crust formation, BD(X) longitude of crust formation, and
BD(Y) latitude of crust formation.
.SH OPTIONAL ARGUMENTS
.TP
OPT(F)IT(polygonfile)
Specify a multisegment closed polygon file that describes the inside area of the grid
where the model should be evaluated; the outside will be set to NaN [Default evaluates model on the entire grid].
.TP
OPT(T)IT(age)
Use a fixed age for model evaluation (i.e., override the ages in the age grid).  This lets you
evaluate the model at a snapshot in time.
#include "explain_-V.txt"
.TP
OPT(:)
Toggles between (longitude,latitude) and (latitude,longitude) input/output.  [Default is (longitude,latitude)].
#include "explain_-bi.txt"
[Default is 2 input columns].
#include "explain_-h.txt"
#include "explain_-icols.txt"
#include "explain_help.txt"
.SH EXAMPLES
We will use a grid with Pacific crust ages (pac_age.nc), a plate motion model (Pac_APM.d),
and a polygon that contains the outline of the present Pacific plate (pac_clip_path.d).
To evaluate the plate motion azimuths at the present time for the Pacific, try
.br
.sp
grdpmodeler pac_age.nc -EPac_APM.d -V -Fpac_clip_path.d -Gpac_dir_0.nc -Sa -T0
.br
.sp
To determine the changes in latitude since crust formation for the entire Pacific, try
.br
.sp
grdpmodeler pac_age.nc -EPac_APM.d -V -Fpac_clip_path.d -Gpac_dlat.nc -Sy
.br
.sp
To determine the plate motion rates in effect when the Pacific crust was formed, try
.br
.sp
grdpmodeler pac_age.nc -EPac_APM.d -V -Fpac_clip_path.d -Gpac_vel.nc -Sr
.br
.sp
To determine how far the crust has moved since formation, try
.br
.sp
grdpmodeler pac_age.nc -EPac_APM.d -V -Fpac_clip_path.d -Gpac_dist.nc -Sd
