#include "common_options.txt"
.TH ISOGMT 1 MANDATE GMT_STRING "Generic Mapping Tools"
.SH NAME
isogmt \- Run GMT command or script in isolation mode
.SH SYNOPSIS
BD(isogmt) IT(command)
.SH DESCRIPTION
BD(isogmt) runs a single BD(GMT) command or shell script in BD(isolation mode).
This means that the files IT(.gmtcommands) and IT(gmt.conf) will be read
from the usual locations (current directory, IT(~/.gmt), or home directory), but
changes will only be written in a temporary directory, which will be removed after
execution. The name of the temporary directory will be available to the command or
script as the environment variable BD(GMT_TMPDIR).
.SH EXAMPLES
Run the shell script IT(script.gmt) in isolation mode
.br
.sp
BD(isogmt) sh script.gmt
.br
.SH "SEE ALSO"
.IR gmt (1),
.IR gmt.conf (5)
