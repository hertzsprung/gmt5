#define CLEARANCE_SI 0.1\fBc\fP/0.1\fBc\fP
#define CLEARANCE_US 0.0254\fBi\fP/0.0254\fBi\fP
.br
\fB+a\fP\fIangle\fP for annotations at a fixed angle, \fB+an\fP for line-normal, or \fB+ap\fP for line-parallel [Default].
.br
\fB+c\fP\fIdx\fP[/\fIdy\fP] sets the clearance between label and optional text box [CLEARANCE_SI (or CLEARANCE_US)].
.br
\fB+f\fP\fIfont\fP sets the desired font [Default \fBANNOT_FONT_PRIMARY\fP].
.br
\fB+g\fP\fIcolor\fP sets color of opaque text boxes [Default is \fBPAGE_COLOR\fP].
.br
\fB+j\fP\fIjust\fP sets label justification [Default is CM].
.br
\fB+k\fP\fIcolor\fP sets color of text labels [Default is \fBCOLOR_BACKGROUND\fP].
#if !defined(CONTOUR)
.br
\fB+l\fP\fIlabel\fP sets the label text.  If "h" we take the label from the current multisegment header (first looking
for \fB\-\fPlabel\fP option, if not the first word after segment flag). If "d" we take the distance along the line
as the label.
#endif
.br
\fB+o|O\fP activates the drawing of rectangular or rounded rectangular outline of text box [Default is no outline].
.br
\fB+p\fP\fIpen\fP sets pen for text box [Default is width = 0.25p, color = black, texture = solid].
.br
\fB+s\fP\fIsize\fP sets the desired font size in points [Default is 9].
.br
\fB+t\fP specifies straight text and transparent rectangular text box [Default].
.br
\fB+T\fP specifies curved text and transparent curved text box following line.
.br
\fB+u\fP\fIunit\fP appends \fIunit\fP to all line labels. If \fIunit\fP starts with a leading hypen (-) then there will be no space
between label value and the unit.
#ifdef GRID
If no \fIunit\fP is appended we use the units listed in the gridfile.
#endif
[Default is no unit].
.br
\fB+^\fP\fIprefix\fP prepends \fIprefix\fP to all line labels. If \fIprefix\fP starts with a leading hypen (-) then there will be no space
between label value and the prefix. [Default is no prefix].
