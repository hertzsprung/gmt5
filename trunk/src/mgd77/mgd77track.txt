.SH NAME
mgd77track \- A shiptrack plotting program
.SH SYNOPSIS
BD(mgd77track) IT(NGDC-ids) GMT_Rgeo_OPT GMT_J_OPT 
[ OPT(A)[BD(c)][IT(size)][,IT(spacing)] ]  [ GMT_B_OPT ] [ OPT(C)BD(f)|BD(g)|BD(e) ] [ OPT(Da)IT(startdate) ] 
[ OPT(Db)IT(stopdate) ] [ OPT(F) ] [ OPT(G)BD(d)|BD(t)IT(gap) ] 
[ OPT(I)IT(ignore) ] [ GMT_K_OPT ] [ OPT(L)IT(trackticks) ] [ GMT_O_OPT ] [ GMT_P_OPT ] [ OPT(Sa)IT(startdist)[BD(u)] ] 
[ OPT(Sb)IT(stopdist)[BD(u)] ] [ OPT(TT)|BD(t)|BD(d)IT(ms),IT(mc),IT(mfs),IT(mf),IT(mfc) ] [ GMT_U_OPT ] 
[ GMT_V_OPT ] [ OPT(W)IT(pen) ] [ GMT_X_OPT ] [ GMT_Y_OPT ] [ GMT_c_OPT ] [ GMT_p_OPT ] [ GMT_t_OPT ]
.SH DESCRIPTION
BD(mgd77track) reads NGDC MGD77 cruises and creates IT(PostScript) code that will plot one or more ship tracks on a
map using the specified projection.  The IT(PostScript) code is written to standard output.

.SH OPTIONAL ARGUMENTS
.TP
OPT(A)[BD(c)][IT(size)][,IT(spacing)]
Append BD(c) to annotate using the MGD77 cruise ID [Default uses the filename prefix].
Optional IT(size) is the font size in points.  The leg annotation font is controlled by BD(FONT_LABEL).
By default, each leg is annotated every time it enters the map region. Alternatively, append ,IT(spacing) to place
this label every IT(spacing) units apart along the track.  Append one of the units BD(k) (km), BD(n) (nautical mile),
BD(d) (day), or BD(h) (hour).
#include "explain_-B.txt"
.TP
OPT(C)BD(f)|BD(g)|BD(e)
Select procedure for along-track distance calculation:
.br
	BD(f) Flat Earth distances.
.br
	BD(g) Great circle distances [Default].
.br
	BD(e) Geodesic distances on current GMT ellipsoid.
.TP
OPT(Da)IT(startdate)
Do not plot data collected before IT(startdate) (yyyy-mm-ddBD(T)[hh:mm:ss])  [Default is first day].
.TP
OPT(Db)IT(stopdate)
Do not plot data collected after IT(stopdate) (yyyy-mm-ddBD(T)[hh:mm:ss]). [Default is last day].
.TP
OPT(F)
Do not apply the error bit flags if present in a MGD77+ file [Default will apply these flags upon reading the data].
.TP
OPT(G)BD(d)|BD(t)IT(gap)
Let successive point separations exceeding BD(d)IT(gap) (km) or BD(t)IT(gap) (minutes) indicate a break
in the track where we should not draw a line [no gaps recognized].  Repeat to use both types of gap checking.
.TP
OPT(I)IT(ignore)
Ignore certain data file formats from consideration. Append BD(a|c|t) to ignore
MGD77 ASCII, MGD77+ netCDF, or plain table files, respectively. The option may
be repeated to ignore more than one format.  [Default ignores none].
#include "explain_-K.txt"
.TP
OPT(L)IT(trackticks)
To put time/distance log-marks on the track. E.g. BD(a)500BD(ka)24BD(ht)6BD(h) means (BD(a))nnotate
every 500 km (BD(k)) and 24 BD(h)(ours), with (BD(t))ickmarks every 500 km and 6 hours.
Alternatively you may use the modifiers BD(d) (days) and BD(n) (nautical miles).
#include "explain_-O.txt"
#include "explain_-P.txt"
.TP
OPT(Sa)IT(startdist)[BD(u)]
Do not plot data that are less than IT(startdist) meter along track from port of departure.
Append BD(k) for km, BD(m) for miles, or BD(n) for nautical miles [Default is 0 meters].
.TP
OPT(Sb)IT(stopdist)[BD(u)]
Do not plot data that are more than IT(stopdist) meter along track from port of departure.
Append BD(k) for km, BD(m) for miles, or BD(n) for nautical miles [Default is end of track].
.TP
OPT(TT)|BD(t)|BD(d)IT(ms),IT(mc),IT(mfs),IT(mf),IT(mfc)
Controls the attributes of the three kinds of markers (BD(T) for the first time marker in a new
day, BD(t) for additional time markers in the same day, and BD(d) for distance markers).  For each
of these you can specify the 5 comma-separated attributes IT(markersize), IT(markercolor),
IT(markerfontsize), IT(markerfont), and IT(markerfontcolor).  Repeat the OPT(T) option for
each marker type.
#include "explain_-U.txt"
.TP
OPT(W)IT(pen)
Append IT(pen) used for the trackline. [Default is 0.25p,black]. 
[Default is solid].

.SH EXAMPLES
To generate a Mercator plot of the track of the cruise 01010007 in the area 70W to
20E, 40S to 20N, using a Mercator scale of SCALE1, label the tracks with 10 points characters, annotate
the boundaries every 10 degrees, draw gridlines every 5 degrees, and mark the track every day and 1000 km,
with ticks every 6 hours and 250 km, and send the plot to the default printer,
enter the following command:
.br
.sp
BD(mgd77track) 01010007 OPT(R)70W/20E/40S/20N SCALE2 OPT(B)10BD(g)5 OPT(A)10 OPT(L)a1da1000kf6hf250k | lpr
