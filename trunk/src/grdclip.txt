#include "common_options.txt"
.TH GRDCLIP 1 MANDATE GMT_STRING "Generic Mapping Tools"
.SH NAME
grdclip \- Clipping of range in grid files.
.SH SYNOPSIS
BD(grdclip) IT(input_file.nc) OPT(G)IT(output_file.nc) [ GMT_Rgeo_OPT ] 
[ OPT(Sa)IT(high/above) ] [ OPT(Sb)IT(low/below) ] [ GMT_V_OPT ] 
.SH DESCRIPTION
BD(grdclip) will set values < IT(low) to IT(below) and/or values > IT(high) to IT(above).  Useful
when you want all of a continent or an ocean to fall into one color or grayshade in image processing,
or clipping of the range of data values is required.  IT(above/below) can be any number or NaN (Not a Number).
You must choose at least one of OPT(Sa) or OPT(Sb).  Use OPT(R) to only extract a subset of the IT(input_file.nc) file.
#include "explain_commonitems.txt"
.SH REQUIRED ARGUMENTS
.TP
IT(input_file.nc)
The input 2-D binary grid file.
.TP
OPT(G)IT(output_file.nc)
IT(output_file.nc) is the modified output grid file.
.SH OPTIONAL ARGUMENTS
#include "explain_-R.txt" 
Using the OPT(R) option will select a subsection of IT(input_file.nc) grid. If this subsection exceeds the
boundaries of the grid, only the common region will be extracted.
.TP
OPT(Sa)IT(high/above)
Set all data[i] > IT(high) to IT(above).
.TP
OPT(Sb)IT(low/below)
Set all data[i] < IT(low) to IT(below).
#include "explain_-V.txt"
#include "explain_help.txt"
#include "explain_grd_inout.txt"
.SH EXAMPLES
To set all values > 70 to NaN and all values < 0 to 0 in file data.nc:
.sp
grdclip data.nc -Gnew_data.nc -Sa70/NaN -Sb0/0 -V
