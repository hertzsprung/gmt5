.TH GMT2DAT l DATE
.SH NAME
gmt2dat \- Convert a binary gmt file to an ASCII file
.SH SYNOPSIS
\fBgmt2dat\fP \fIgmtfile\fP
.SH DESCRIPTION
gmt2dat reads a <legid>.gmt file and converts it to a regular ASCII text file that can be edited
using the text editor. The ASCII text is sent to standard output and can be redirected to a
file/program/printer etc.
.SH EXAMPLES
To edit the data in c2104.gmt, try
.br
.sp
gmt2dat c2104.gmt > c2104.ascii
