GMT for Windows


We offer three Windows Installers for those who do not wish to compile from source:

Required: The GMT basic installer will install all executables (including the supplements), the crude, low, and intermediate GSHHS coastline data, all documentation in HTML format, and the example batch scripts and test data [14.6 Mb].

Optional 1: The GSHHS installer will install the optional high and full resolution GSHHS v1.10 coastlines [36.8 Mb].
Optional 2: The PDF installer will install the documentation in PDF format [10.9 Mb].

Should you have additional needs for source code or tutorial data sets, etc., you may download the complete GMT distribution as bzipped tarballs from any of our GMT ftp sites and extract the files with a suitable extractor (such as 7-zip).
