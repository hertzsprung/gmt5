.SH NAME
sphinterpolate \- Spherical gridding in tension of data on a sphere
.SH SYNOPSIS
BD(sphinterpolate) [ IT(table) ] OPT(G)IT(grdfile) [ GMT_I_OPT ] 
[ OPT(Q)IT(mode)[/IT(options)] ] [ GMT_Rgeo_OPT ] [ GMT_V_OPT ] [ OPT(Z) ] 
[ GMT_b_OPT ] [ GMT_h_OPT ] [ GMT_i_OPT ] [ GMT_r_OPT ] [ GMT_colon_OPT ]
.SH DESCRIPTION
BD(sphinterpolate) reads one or more ASCII [or binary] files (or standard input) containing lon, lat, f and
performs a Delaunay triangulation to set up a spherical interpolation in tension.  The final grid is saved
to the specified file.  Several options may be used to affect the outcome, such as choosing local versus global gradient
estimation or optimize the tension selection to satisfy one of four criteria.
#include "explain_commonitems.txt"
.SH REQUIRED ARGUMENTS
.TP
OPT(G)IT(grdfile)
Name of the output grid to hold the interpolation.
.SH OPTIONAL ARGUMENTS
#include "explain_intables.txt"
#include "explain_-I.txt"
.TP
OPT(Q)IT(mode)[/IT(options)]
Specify one of four ways to calculate tension factors to preserve local shape properties or
satisfy arc constraints [Default is no tension].
.TP
OPT(Q)0
Piecewise linear interpolation; no tension is applied.
.TP
OPT(Q)Q1
Smooth interpolation with local gradient estimates.
.TP
OPT(Q)Q2
Smooth interpolation with global gradient estimates.  You may optionally append /IT(N)/IT(M)/IT(U), where
IT(N) is the number of iterations used to converge at solutions for gradients when variable tensions are selected (e.g., OPT(T) only) [3],
IT(M) is the number of Gauss-Seidel iterations used when determining the global gradients [10], and
IT(U) is the maximum change in a gradient at the last iteration [0.01].
.TP
OPT(Q)Q3
Smoothing.  Optionally append IT(/E/U) [/0/0], where
IT(E) is Expected squared error in a typical (scaled) data value, and
IT(U) is Upper bound on  weighted sum of squares of deviations from data.
