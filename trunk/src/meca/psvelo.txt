.TH PSVELO 1 MANDATE GMT_STRING "Generic Mapping Tools [meca supplement]"
.SH NAME
psvelo \- Plot velocity vectors, crosses, and wedges on maps
.SH SYNOPSIS
.br
BD(psvelo) [ IT(datatables) ] GMT_J_OPT GMT_Rgeo_OPT [ GMT_B_OPT ] 
[ OPT(D)IT(sigma_scale) ] [ OPT(F)IT(color) ] 
[ OPT(E)IT(color) ] [ OPT(G)IT(fill) ] [ GMT_K_OPT ] [ OPT(L) ] [ OPT(N) ] 
[ GMT_O_OPT ] [ GMT_P_OPT ] [ OPT(S)IT(symbol)/IT(scale)/IT(conf)/IT(font_size) ] [ GMT_U_OPT ] 
[ GMT_V_OPT ] [ OPT(W)IT(pen) ] [ GMT_X_OPT ] [ GMT_Y_OPT ] [ GMT_c_OPT ] [ GMT_h_OPT ] [ GMT_i_OPT ] [ GMT_colon_OPT ]
.SH DESCRIPTION
.br
BD(psvelo) reads data values from IT(files) [or standard input]
and generates IT(PostScript) code that will plot 
velocity arrows on a map.  Most options are the same as for BD(psxy),
except OPT(S).  The IT(PostScript) code is written to standard
output. The previous version (BD(psvelomeca)) is now obsolete. It has
been replaced by BD(psvelo) and BD(psmeca).
.br
.sp

OPT(S)Selects the meaning of the columns in the data file and the
figure to be plotted.
.br
.sp
.TP
OPT(Se)IT(velscale/confidence/fontsize).
Velocity ellipses in
(N,E) convention.  IT(Vscale) sets the scaling of the velocity arrows.
This scaling gives inches (unless BD(c), BD(i), BD(m), or BD(p) is appended).
IT(Confidence) sets the 2-dimensional confidence limit for
the ellipse, e.g., 0.95 for 95% confidence ellipse.  IT(Fontsize)
sets the size of the text in points.  The ellipse will be filled with
the color or shade specified by the OPT(G) option [default
transparent].  The arrow and the circumference of the ellipse will be
drawn with the pen attributes specified by the OPT(W) option.
Parameters are expected to be in the following columns:
.TP
.B 1,2
longitude, latitude of station (OPT(:) option interchanges order)
.TP
.B 3,4
eastward, northward velocity (OPT(:) option interchanges order)
.TP
.B 5,6
uncertainty of eastward, northward velocities (1-sigma) (OPT(:)
option interchanges order)
.TP
.B 7
correlation between eastward and northward components
.TP
.B 8
name of station (optional).
.br
.sp
.TP
OPT(Sn)IT(barscale.)
Anisotropy bars.  IT(Barscale) sets the scaling of the bars
This scaling gives inches (unless BD(c), BD(i), BD(m), or BD(p) is appended).
Parameters are expected to be in the following columns:
.TP
.B 1,2
longitude, latitude of station (OPT(:) option interchanges order)
.TP
.B 3,4
eastward, northward components of anisotropy vector (OPT(:) option interchanges order)
.br
.br
.sp
.TP
OPT(Sr)IT(velscale/confidence/fontsize)
Velocity ellipses in rotated convention.  IT(Vscale) sets the scaling of the velocity
arrows. This scaling gives inches (unless BD(c), BD(i), BD(m), or BD(p) is appended).
IT(Confidence) sets the 2-dimensional confidence limit for
the ellipse, e.g., 0.95 for 95% confidence ellipse.  IT(Fontsize)
sets the size of the text in points.
The ellipse will be filled with the color or shade specified by the OPT(G) option
[default transparent].  The arrow and the circumference of the ellipse will
be drawn with the pen attributes specified by the OPT(W) option.
Parameters are expected to be in
the following columns:
.TP
.B 1,2
longitude, latitude, of station (OPT(:) option interchanges order)
.TP
.B 3,4
eastward, northward velocity (OPT(:) option interchanges order)
.TP
.B 5,6
semi-major, semi-minor axes
.TP
.B 7
counter-clockwise angle, in degrees, from horizontal axis to major axis of ellipse.
.TP
.B 8
name of station (optional)
.br
.sp
.TP
OPT(Sw)IT(wedge_scale/wedge_mag).
Rotational wedges.
IT(Wedge_scale) sets the size of the wedges in inches
(unless BD(c), BD(i), BD(m), or BD(p) is appended).  Values are
multiplied by IT(Wedge_mag) before plotting.  For example, setting
IT(Wedge_mag) to 1.e7 works well for rotations of the order of 100
nanoradians/yr.  Use OPT(G) to set the fill color or shade for the
wedge, and OPT(E) to set the color or shade for the uncertainty.
Parameters are expected to be in the following columns:
.TP
.B 1,2
longitude, latitude, of station (OPT(:) option interchanges order)
.TP
.B 3
rotation in radians
.TP
.B 4
rotation uncertainty in radians
.br
.sp
.TP
OPT(Sx)IT(cross_scale)
gives Strain crosses.  IT(Cross_scale) sets the
size of the cross in inches (unless BD(c), BD(i), BD(m), or BD(p) 
is appended).  Parameters are expected to be in the
following columns:
.TP
.B 1,2
longitude, latitude, of station (OPT(:) option interchanges order)
.TP
.B 3
eps1, the most extensional eigenvalue of strain tensor, with extension taken positive.
.TP
.B 4
eps2, the most compressional eigenvalue of strain tensor, with extension taken positive.
.TP
.B 5
azimuth of eps2 in degrees CW from North.
.br
.sp
.br
.SH OPTIONAL ARGUMENTS
.TP
OPT(A)
IT(Arrow_width/Head_length/Head_width )
Size of arrow in inches.  [Default is 0.03/0.12/0.09].
#include "explain_-B.txt"
.TP
OPT(C)
Offset focal mechanisms to the latitude and longitude specified in the last
two columns of the input file.
.br
.TP
OPT(D)IT(Sigma_scale )
can be used to rescale the uncertainties of velocities (OPT(Se) and
OPT(Sr)) and rotations (OPT(Sw)).  Can be combined with the
IT(confidence ) variable.
.br
.TP
OPT(F)IT(fill)
Sets the color or shade used for frame and annotation. [Default is black]
.br
.TP
OPT(E)IT(fill)
Sets the color or shade used for filling uncertainty wedges (OPT(Sw))
or velocity error ellipses (OPT(Se) or OPT(Sr)).  [If OPT(E) is not
specified, the uncertainty regions will be transparent.]
.br
.TP
OPT(G)IT(fill)
Select filling of ellipses, wedges, and focal mechanisms.  By
convention, the compressional quadrants of the focal mechanism beach
balls are shaded.  Set the color [Default is
black].  Optionally, specify OPT(Gp)IT(icon_size/pattern), where
IT(pattern) gives the number of the image pattern (1-90) OR the name
of a icon-format file.  IT(icon_size) sets the unit size in inches.
To invert black and white pixels, use OPT(GP) instead of
OPT(Gp).  See BD(pspatterns) for information on individual
patterns.
#include "explain_-K.txt"
.TP
OPT(L)
Draw lines.  Ellipses and fault planes will have their outlines drawn using current pen (see OPT(W)).
.br
.TP
OPT(N)
Do BD(NOT) skip symbols that fall outside the frame boundary specified by OPT(R).
 [Default plots symbols inside frame only].

Set pen attributes for velocity arrows, ellipse circumference and fault plane edges.
 [Defaults: width = 0.25p, color = black, style = solid].

.SH EXAMPLES
.sp
The following should make big red arrows with green ellipses, outlined
in red.  Note that the 39% confidence scaling will give an ellipse which
fits inside a rectangle of dimension Esig by Nsig.
.br
.sp
BD(psvelo) << END OPT(H)2 OPT(R)-10/10/-10/10 OPT(W)0.25p,red
OPT(G)green OPT(L) OPT(Se)0.2/0.39/18 OPT(B)1g1 OPT(Jx)0.4/0.4
OPT(A)0.1/0.3/0.3 OPT(P) OPT(V)  > test.ps
