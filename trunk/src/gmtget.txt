#include "common_options.txt"
.TH GMTGET 1 MANDATE GMT_STRING "Generic Mapping Tools"
.SH NAME
gmtget \- To list a single BD(GMT) parameter
.SH SYNOPSIS
BD(gmtget) [ OPT(G)IT(defaultsfile) ] [ OPT(L) ] IT(PARAMETER1) [ IT(PARAMETER2) IT(PARAMETER3) ... ]
.SH DESCRIPTION
BD(gmtget) will list the value of one or more BD(GMT) default parameters.
.SH REQUIRED ARGUMENTS
.TP
PARAMETER
Provide one or several parameters of interest.  The current value of those parameters
will be writen to IT(stdout).  For a
complete listing of available parameters and their meaning, see the BD(gmt.conf) man page.
.SH OPTIONAL ARGUMENTS
.TP
OPT(G)IT(defaultsfile)
Name of specific BD(gmt.conf) file to read [Default looks first in current directory, then in your
home directory, then in ~/.gmt and finally in the system defaults].
.TP
OPT(L)
Return the values of the parameters on separate lines [Default returns all selected parameter values on one line separated by spaces]
.SH EXAMPLE
To list the value of the parameter PS_COMMENTS:
.br
.sp
BD(gmtget) PS_COMMENTS
To get both the values of the parameter GRID_CROSS_SIZE_PRIMARY and GRID_CROSS_SIZE_SECONDARY on one line, try
.br
.sp
BD(gmtget) GRID_CROSS_SIZE_PRIMARY GRID_CROSS_SIZE_SECONDARY
.SH "SEE ALSO"
.IR gmt (1),
.IR gmt.conf (5),
.IR gmtset (1)
